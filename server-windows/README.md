# Server Windows App

This folder contains public-safe server app references, assets, and future source/package files.

## Folders

- package-reference: Safe reference files from the tray/server package
- assets: App icons and branding assets
- web-assets: Browser UI hero and icons
- scripts: Future sanitized server scripts

Private runtime data, logs, generated output, local configs, inventories, and binaries are intentionally excluded.
