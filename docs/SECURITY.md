# Security Policy

This public repository must never contain private runtime data or secrets.

## Never commit

- Real API keys
- Real PINs
- SMB credentials
- Private IP addresses
- Private Tailscale hostnames
- Real media inventory CSV or JSON files
- Logs
- OUTPUT folders
- APKs
- EXE builds unless intentionally released later
- Personal configs
- Generated clips, screenshots, or audio from private media

## Safe placeholders

- http://home-server.local:8787
- http://tailscale-device:8787
- bmr_demo_xxxxxxxxx
- \\MEDIA-SERVER\Movies
- \\MEDIA-SERVER\tv
